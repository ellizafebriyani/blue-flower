This project is an interactive web application developed using **HTML, CSS, and JavaScript** that combines animation, visual effects, and music to create an engaging user experience. The website begins with a welcome screen displaying the text **"HI THERE!"**, where each letter lights up and fades alternately to create a blinking effect that captures the user's attention. A **"Tap Here"** button is provided to initiate the main animation. Once clicked, the page transitions into a beautiful night-themed scene where flowers gradually grow and bloom through smooth animations, accompanied by background music that enhances the overall atmosphere.
